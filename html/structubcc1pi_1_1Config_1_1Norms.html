Detailed Description

The sample normalisations structure.

These values can be found by:

  • Running the ubcc1pi::CountPOT macro (for overlays, dirt and detector variation files)
  • Running the ubcc1pi::GetRunSubrunList macro (for data), and then running Zarko's tool: /uboone/app/users/zarko/getDataInfo.py -v2 –run-subrun-list runSubrunList.txt
