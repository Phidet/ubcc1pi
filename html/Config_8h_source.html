424  * to one. In this case, the set of parameters can be "combined" into one parameter (with the user-defined name). The value
425  * of the combined parameter is taken from whichever parameter in the set has been varied(or equivalently just the product

430  * a weight of one for all other channels. In this sense, there is a "correlation" between the weights for each channel.
