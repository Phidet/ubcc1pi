Detailed Description

template<typename R, typename... Args>
class ubcc1pi::CrossSectionHelper::SystCacheFunction< R, Args >

A wrapper around an arbitrary function that caches the result of the function in a SystPartialMap. If the function has already been executed for a given systematic parameter / universe then the a call to the function will return the cached value instead. This is provides a helful way of storing the result of a function that we might want to use in multiple places. This is done for performance reasons, so please excuse the added complexity!
