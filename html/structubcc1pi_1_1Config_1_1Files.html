The detector variation files.

The events in each detector variation sample (in the same run) are identical apart from the detector parameter that's been changed - this way we can limit the effects of statistical variations. As a result we also need to have a central-value (CV) sample in which nothing is changed that we can compare to. Here, only some of the variations are available for run 1. Instead we use run3b, for which the varaitions are available. In every case the variation is considered wrt to the relevant "CV" file, and the fractional difference is quantity we care about.
